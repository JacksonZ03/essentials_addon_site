A Set of Essential Features Missing in Vanilla Blender

Gravity Snap

Perfectly snap any mesh or empty object to the surface of another mesh object instantly - just as if you dropped it.

Great for quickly aligning meshes in your scene.

  • Respects object size, shape and orientation
  • Origin point moves with the object
  • Custom offset adjustment allow you to embed an object just above or below of target surface

Auto-Highlight Selection in Outliner

While Blender does have a button to expand collections in Outliner to show the selected object, it requires an extra button press each time.

With this addon, the process is automated, saving you both time and effort when navigating a large scene with many objects.

Center Camera Perspective

Quickly center the camera perspective view with a single click.

  • Vanilla Blender does not provide this feature at all
  • Closest thing in vanilla Blender is pressing Home, but that zooms camera to fullscreen

Spacebar Play/Pause Toggle

Prevent accidental timeline playback when using spacebar for other applications.

Now you can play/pause YouTube tutorials with multiple screens open, without accidentally playing your animation.

  • Works in all animation workspaces (timeline, graph editor, etc)
  • If keymap is not bound to spacebar, the button doesn't show in the UI

NEW UPDATE v1.1.0:

Viewport/Render Visibility Sync

Sync the viewport and render visibility with a single click.

MASSIVE timesaver when working on large scenes as manually searching and toggling visibility of each object often eats away hours of accumulated time.

Don't need all the features?

Enable or disable them in the addon preferences. This removes all buttons and toggles related to that feature from the UI.

Change preferences through:

  1. Edit > Preferences > Add-ons
  2. Find "Essentials" in the add-ons list
  3. Expand the addon preferences
